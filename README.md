## ⚡ Quick Start

1. **Initialize:** Run `/pdes` to boot the system.
2. **Context:** Drop any raw data files into the `input/` folder.
3. **Diagnose:** Run `/perceive [area]` to scan your current state.
4. **Upgrade:** Load your recommended level via `/level g[x]`.
5. **Execute:** Follow the 6-Phase pipeline to generate trackers in `output/`.
